# نظام الاتصالات الداخلي - قسم الإعداد والتدريب

هذا النظام هو تطبيق ويب مخصص لإدارة البريد الإلكتروني الداخلي والتعميمات بين مسؤولي الشعب والموظفين داخل الشبكة المحلية (Local Network) أو الخادم المركزي.

## مميزات النظام:
- **تأمين البيانات:** يستخدم النظام تقنية تشفير كلمات المرور (bcrypt) ورموز مصادقة الجلسات (JWT) لمنع التلاعب.
- **شبكة محلية (Intranet):** يمكن تشغيل النظام على حاسوب مركزي (Server) ليقوم الموظفون بالدخول إليه عبر متصفحات الإنترنت الخاص بهم عبر الشبكة.
- **تخزين محلي مرن:** يتم تخزين البيانات بشكل مؤمن في ملف `data_store.json` لسهولة أخذ النسخ الاحتياطية.

---

## 🚀 كيفية التشغيل في بيئة العمل (الإنتاج)

لقد قمنا بتوفير نصوص تشغيلية (Batch Files) لتسهيل العمل على أنظمة ويندوز:

1. انقر نقراً مزدوجاً على الملف **`start_server.bat`**.
2. ستظهر نافذة أوامر (CMD) سوداء تخبرك بأن الخادم يعمل على منفذ `3000`.
3. لا تقم بإغلاق هذه النافذة (فهي التي تُبقي النظام قيد التشغيل).
4. الآن، يمكن لأي موظف متصل بالشبكة (الراوتر) فتح متصفحه وكتابة الـ IP الخاص بحاسوب الخادم، مثلاً:
   `http://192.168.1.50:3000`

## 📦 كيفية أخذ نسخة احتياطية من البيانات

لضمان عدم ضياع الرسائل والتعميمات:
- انقر نقراً مزدوجاً على الملف **`backup_database.bat`**.
- سيقوم هذا الملف فوراً بإنشاء مجلد يُسمى `backups` ونسخ قاعدة البيانات داخله مع وضع التاريخ والوقت في اسم الملف (مثال: `data_store_backup_2026-06-05_1030.json`).

---

## 🛠️ للمطورين (Development)

إذا كنت مطوراً وترغب بالتعديل على النظام:
1. قم بفتح محرر الأكواد.
2. قم بتثبيت الحزم: `npm install`
3. قم بتشغيل وضع التطوير: `npm run dev`
4. لعمل بناء نهائي (Build) قبل النشر للإنتاج: `npm run build`
